# Giai thừa

Trong toán học, giai thừa của một số nguyên không âm `n` được ký hiệu là `n!`, có giá trị bằng tích các số nguyên dương nhỏ hơn hoặc bằng `n`. Ví dụ :

```
5! = 5 * 4 * 3 * 2 * 1 = 120
```

| 3   |                 6 |
| 4   |                24 |
| 5   |               120 |
